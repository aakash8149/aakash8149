## Featured Project

### Smart AI-Powered Banking System
> *Project — Production-Grade Backend with Embedded AI Modules*

A full-scale banking backend built from scratch, with Python AI modules embedded directly into the Spring Boot service layer — not as an external sidecar, but as a core architectural component influencing real-time business logic.

**Engineering decisions:**

- Integrated a **Python NLP module** for automated query classification and customer intent resolution directly within the service layer
- Built a **real-time fraud detection pipeline** using Scikit-learn, triggered at the transaction service level with live scoring
- Implemented **Text-to-SQL automation** translating natural language into optimized database queries
- Secured the entire system with **Spring Security + JWT** and granular role-based access control
- Containerized with **Docker**; deployed via **Railway** with a full CI/CD pipeline

**Results**

```
60% reduction in manual processing overhead
30–40% improvement in API response time
High-precision fraud classification at transaction layer
```

**Stack**

```
Java  ·  Spring Boot  ·  Spring Security  ·  Python  ·  Scikit-learn
PostgreSQL  ·  Docker  ·  Railway  ·  REST APIs  ·  JWT  ·  CI/CD
```
